Sobre el Proyecto

La Visión del Creador

Ricardo Montes Carrasco

Este proyecto fue creado para la clase de 1º DAW, en el módulo de programación, con el proposito de crear una base de datos relacional en XAMPP y acceder a ella, utilizando lenguajes HTML, CSS, JS y Javascript.

La base de datos subyacente ha sido diseñada para capturar cada aspecto de este mundo: desde la geografía física de los planetas hasta las estadísticas detalladas de cada personaje y criatura que los habita.
